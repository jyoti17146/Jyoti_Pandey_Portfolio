## ✨ Features

- **Responsive Design:** Fully optimized for seamless viewing across desktops, tablets, and mobile devices.
- **Embedded Stylings:** Highly customized UI built from scratch using modern CSS grid, flexbox layout structures, and fluid typography.
- **Multi-disciplinary Showcase:** Dedicated sections detailing my engineering skills, development projects, literary works, and educational milestones.
- **Sleek Aesthetics:** A modern, dark-themed user interface accented with premium gold and deep tones to enhance scannability and readability.

---

## 🛠️ Technologies Used

* **HTML5:** Structuring the semantic layout and web components.
* **CSS3:** Custom properties (variables), grid configurations, flexbox positioning, and micro-interactions.
* **Google Fonts:** Integration of elegant typography (`Space Grotesk` and `Cormorant Garamond`).
